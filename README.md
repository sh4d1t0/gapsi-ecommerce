## Gapsi Ecommerce

1.  **Clona el repositorio.**

Una vez creado el repositorio procede a instalar

```shell
# ejemplo del comando para instalar
npm install
```
2.  **Iniciar el entorno de desarrollo.**

Navega hasta el directorio de tu sitio e inicia el ambiente.

```shell
cd gapsi-ecommerce/
npm run dev
```
